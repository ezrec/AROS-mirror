For users

  • What is BGUI?

    • BGUI stands for BOOPSI Graphical User Interface. BGUI is free GUI toolkit for the Amiga OS.

      BGUI comes with:

    • Software development kit for programming Amiga OS applications in C and E languages.
    • GUI builder
    • Documentation in Autodoc, AmigaGuide and HTML formats.
    • Documentation editor
    • Library to create BGUI programs in ARexx
    • Preferences editor to customize BGUI programs look and feel.

For developers

  • Joining BGUI developers team

    • Would you like to join the BGUI developers team to help fixing existing bugs or add new features?

    BGUI has been integrated in the AROS project (Amiga Research Operating System project). If you would like you help in BGUI development, join AROS (<http://www.aros.org/>).
